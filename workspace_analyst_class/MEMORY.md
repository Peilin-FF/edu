# Core Conceptual Model

All sensors follow the same basic structure:

Physical quantity  
→ sensitive element  
→ conversion element  
→ signal conditioning  
→ electrical signal output

Typical components:

- sensitive element
- transduction element
- signal conditioning circuit

Understanding this **measurement chain** is fundamental.

# Key Knowledge Points

## Sensor Fundamentals

Core topics:

- sensor definition
- sensor classification
- static characteristics
- dynamic characteristics
- sensor selection principles

Important concepts:

Static characteristics

- accuracy
- linearity
- repeatability

Dynamic characteristics

- response time
- frequency response

Selection principles

- suitability
- reliability
- compatibility
- cost effectiveness

Typical difficulty:

Students confuse **static accuracy** with **dynamic response capability**.

---

## Pressure Sensors

Main types:

- strain gauge sensors
- piezoelectric sensors
- inductive sensors

Core principles:

strain effect  
piezoelectric effect  
electromagnetic induction

Important engineering issues:

- temperature compensation
- signal conditioning
- measurement error

Typical misconception:

Students often believe **higher precision sensors are always better**, ignoring application requirements and cost.

# Common Student Difficulties

Typical learning problems observed in this course:

### Conceptual understanding

Students often:

- memorize sensor types without understanding physical principles
- confuse similar sensing mechanisms

### Engineering reasoning

Students struggle with:

- selecting appropriate sensors for real scenarios
- analyzing measurement errors
- designing signal conditioning circuits

### Application thinking

Students may:

- focus on theory but ignore engineering constraints
- overlook cost-performance tradeoffs

# Analysis History & Patterns

## 2022-10-25 Exam Analysis (First Report)

### Key Findings
- **Class size**: 5 students
- **Average score**: 77.2/100
- **Pass rate**: 100%
- **Score range**: 61-84 (23-point gap)

### Persistent Weaknesses Identified
1. **Strain effect (应变效应)**: 0% mastery rate - critical weakness
2. **Sensor definition (传感器定义)**: 20% mastery rate - fundamental concept issue
3. **Moisture content definition (含水量定义)**: 40% mastery rate - terminology confusion

### Learning Patterns Observed
- Students perform better on memorization-based questions
- Application-oriented questions show higher error rates
- Emerging technology topics (IoT, vehicular networks) are challenging
- Material properties (piezoelectric, thermistor) are frequently confused

### Teaching Implications
1. Need for clearer conceptual explanations of strain effect
2. Reinforcement of sensor fundamentals required
3. More practical application exercises needed
4. Differentiated instruction to address individual learning gaps
