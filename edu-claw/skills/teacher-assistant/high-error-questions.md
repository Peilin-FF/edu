# 高错率题目分析

汇总所有学生的错题，找出全班共性问题。

## 数据来源
- 每个学生的题目列表 → 统计每道题的错误人数/总人数

## 分析维度
- 按错误率排序（错误人数/答题人数）
- 按知识点聚合（哪个知识点集中出错）
- 按题型分类（选择题/填空题/判断题哪类错最多）

## 回答模板
- "错误率最高的题目是第 X 题（XX%的同学答错），知识点：XX"
- "填空题整体错误率（XX%）高于选择题（XX%），建议加强概念理解"
