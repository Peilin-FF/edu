# 班级知识掌握分析

从班级所有学生的考试数据中，汇总每个知识点的平均掌握度。

## 数据来源
- 每个有数据的学生的 masteryMap → 汇总为 classMastery
- 包含：知识点名称、平均掌握度、总错题数、涉及学生数

## 分析维度
| 维度 | 方法 |
|------|------|
| 严重薄弱 | avgMastery < 50% |
| 需要巩固 | 50% ≤ avgMastery < 80% |
| 已掌握 | avgMastery ≥ 80% |

## 回答模板
- "班级在 XX 知识点平均掌握度只有 XX%，建议重点复习"
- "以下 3 个知识点全班错误率最高：..."
