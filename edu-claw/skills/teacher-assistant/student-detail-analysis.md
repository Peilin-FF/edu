# 个别学生分析

查看某个学生的详细考试情况、错题、知识掌握度。

## 数据来源
- 学生考试数据 JSON（总分、题目列表）
- 每道题的得分、错因分析

## 分析维度
- 该学生的总分和班级排名
- 该学生的薄弱知识点
- 该学生的具体错题和错因
- 与班级平均水平的对比

## 回答模板
- "XX同学总分 XX 分，班级排名第 X"
- "主要薄弱点在 XX 和 XX，建议..."
